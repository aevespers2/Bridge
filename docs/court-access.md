# Court access handout

Use the QR-linked lookup as a public access and verification tool, not as an allegation package.

## QR target

```text
https://aevespers2.github.io/Bridge/property-lookup/standalone.html
```

## Presentation language

> This QR code links to a public-record lookup that helps users identify which official property records may need to be pulled or verified. It does not ask the viewer to accept any allegation as fact.

## Boundary

This tool is not legal advice, a title opinion, a valuation, an ownership determination, or a misconduct finding. Official assessor, recorder, tax, city, court, and GIS records control.

## Current snapshot

- Records: 19,442
- Search fields: APN, situs/address text, city, use class
- Public lookup file: `/property-lookup/standalone.html`
