# Public property record lookup

{: .lead}
Search a public parcel snapshot by APN, address text, city, or use class. The lookup shows record-pull priorities and verification signals without treating any claim as proved.

## What the QR opens

<div class="grid">
  <section class="card">
    <h3>Public lookup</h3>
    <p>Anyone with the QR can search the current public snapshot from a browser.</p>
  </section>
  <section class="card">
    <h3>Verification first</h3>
    <p>Rows identify official record categories to pull or verify before drawing conclusions.</p>
  </section>
  <section class="card">
    <h3>Evidence boundary</h3>
    <p>The public site excludes owner fields and keeps hypotheses separate from observed records.</p>
  </section>
  <section class="card">
    <h3>Evidence graph</h3>
    <p>Public review page shows observations, claims, validation status, source requirements, and reproducibility commands.</p>
  </section>
  <section class="card">
    <h3>UVIP/QSO mesh</h3>
    <p>A colorful deterministic state signature carries lookup, attestation, and evidence-boundary metadata.</p>
  </section>
</div>
